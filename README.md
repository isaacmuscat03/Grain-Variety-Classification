# Classification of grain varieties using hyperspectral imaging and deep learning
This is the repository for the thesis project with the above title

Currently, the development plan is the following:
- Pre-processing pipeline
- Proper data split
- Examine traditional ML techniques - starting simple with SVMs, k-NNs and Random Forests, narrowing the specific models wherever preferable
- Construct Deep Learning CNN models - exact model not yet determined

The goal is to classify images of different grains as accurately as possible
